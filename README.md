# Studentkatalog Agile Grupp5

Ett TypeScript-baserat projekt för att hantera en studentkatalog.

## Projektstruktur

```
student_katalog/
├── index.html          # Huvudsida med HTML-struktur
├── style.css           # CSS-styling
├── src/
│   └── main.ts         # TypeScript huvudfil
├── tsconfig.json       # TypeScript konfiguration
├── package.json        # Node.js projektfil
└── README.md          # Detta dokument
```

## Installation

1. Installera Node.js dependencies:

```bash
npm install
```

2. Kompilera TypeScript:

```bash
npm run build
```

3. Starta utvecklingsserver:

```bash
npm start
```

## Utveckling

- `npm run dev` - Kompilera TypeScript i watch-läge
- `npm run build` - Kompilera TypeScript
- `npm start` - Starta lokal server på port 3000

## Features

- TypeScript för typsäkerhet
- Responsiv design
- Modulär kod struktur
- Enkel att utöka med nya funktioner
